### Quick Start

#### Prerequisites
- **Rust 1.75+** (https://rustup.rs/)
- **Node.js 18+** (https://nodejs.org/)
- **npm** (comes with Node.js)
- **Docker** (optional, for containerized deployment)

#### Build & Test Everything
```bash
# Build blockchain node, dashboard, and run all tests
./build_and_test.sh          # Linux/macOS
build_and_test.bat          # Windows

# Or manually:
cargo build --release
cd dashboard && npm install && npm run build && cd ..
```

#### Start Complete System
```bash
# Start blockchain node + dashboard
./dist/start_all.sh         # Linux/macOS
dist\start_all.bat         # Windows

# Access points:
# - Dashboard: http://localhost:3000
# - Node RPC:  http://localhost:9944
# - Node WS:   ws://localhost:30333
```

#### Individual Components
```bash
# Start only blockchain node
./dist/start_node.sh        # Linux/macOS
dist\start_node.bat        # Windows

# Start only dashboard
./dist/start_dashboard.sh   # Linux/macOS
dist\start_dashboard.bat   # Windows
```
